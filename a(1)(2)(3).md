### 分析

1. 上一次运行后返回了函数接受新参数.
2. 最终函数返回还是函数，需要通过修改其valueOf, toString 外界才能获取最终结果 

### 代码

```

function a(n){
    let t = function(y){return a(n + y)}
    t.valueOf = t.toString = function(){
        return n
    }
    return t
}

let result = a(1)(2)(3)(4)
console.log(result)
//10
```
