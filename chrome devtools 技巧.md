```
1. command 命令
command + p 查找文件
command + shift + p 执行命令（dom或页面截图等功能）

2. 重新发起xhr请求

network panel 的请求右键 Replay XHR 可以再次发送这个请求，很方便！

3. copy 命令
很有用！！
console内输入copy(xxx)方法就可以拷贝js对象
比如copy(location), 就可以去其他地方粘贴拷贝的页面location对象了！

4. 自定义常用代码片段
在source下有 Snippets 标签, 可以添加常用代码，下次就可以直接使用了。

5. keys, values命令可以直接打出对象键列表或值列表

```
