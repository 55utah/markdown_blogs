#### 数组去重，数组平铺
去重：各种方法，及其性能情况
```
以15W数据测试时间；
1.两个for循环，外层遍历，内存判断是否重复；
性能很差。

2.排序+一次遍历相邻是否相等
一次排序，一次遍历，性能较好；
耗时120ms

function distinct(a) {
    let arr = [].concat(a)
    let result = [arr[0]]
    arr.sort((a, b) => {return a - b})
    for (let i = 1;i < arr.length; i++) {
        if(arr[i] != arr[i - 1]) result.push(arr[i])
    }
    return result
}

3.new Set()

return Array.from(new Set(a))

耗时：57ms

4.for..or + object
利用对象属性不重复特点，检验元素是否重复；

function distinct(a){
    let arr = [].concat(a)
    let obj = {}
    let result = []
    for(let i of arr){
        if(!obj[i]){
            obj[i] = i
            result.push(i)
        }
    }
    return result
}
耗时16ms

所以日常使用的[...new Set(a)] 效率也是很好的，第四种是效率最高的方法；

###### 数组平铺
分布用递归，深度优先遍历（循环）来做
```
递归:

function flat(a){
    for(let i of a){
        return Array.isArray(i) ? flat(i) : i
    }
}

循环:

function flat(a){
    let p = a, r = []
    while(p.length > 0){
        let ca = p.shift()
        if(Array.isArray(ca)){
            for(let i = ca.length - 1; i >= 0; i --){
                p.unshift(ca[i])
            }
        }else{
            r.push(ca)
        }
    }
    return r
}
